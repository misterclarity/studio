# **App Name**: ExhibitAI

## Core Features:

- Image Upload & Analysis: Allows users to upload images of museum exhibits for AI analysis. This system detects distinct artifacts and analyzes relevant technical and cultural metadata about their properties.
- Metadata Extraction & Storage: AI extracts item metadata (size, era, company, materials, weight, topics, etc.) and stores the information in the system. Any photo or additional description for an artifact enhances metadata collection to keep a centralized dataset.
- Item Catalog: Maintains a unified item collection for managing information on artifacts. Uses uploaded photos and their metadata for cataloguing. 
- Advanced Search Functionality: Implements a robust search feature enabling users to find artifacts by specifying any data category or data combination to return relevant catalog matches. 
- AI Contextual Chat: Lets users chat with an AI tool that answers cultural, historical, and technological context of the museum's collection.

## Style Guidelines:

- Primary color: Deep crimson (#800020) to evoke sophistication, history, and curiosity.
- Background color: Soft beige (#F5F5DC) to provide a neutral backdrop that emphasizes the artifacts' details.
- Accent color: Gold (#FFD700) to highlight interactive elements and create visual interest.
- Headline font: 'Playfair', serif, to convey a classic aesthetic
- Body font: 'PT Sans', sans-serif, for clean readability.
- Minimal, classic icons inspired by museum signage.
- Clean, gallery-like display with ample white space, ensuring that the focus remains on the artifacts and information.
- Gentle fade-in animations and transitions to enhance the user experience without being distracting.
